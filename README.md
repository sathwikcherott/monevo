# Monevo

A minimal savings challenge app focused on calm progression, consistency, and motivation.

Monevo is designed to help users save money without the clutter and overwhelm of traditional finance apps. Instead of complicated dashboards and noisy analytics, the app focuses on visual progress, milestone psychology, and a premium user experience.

---

# Features

## Savings Goal Tracking
- Set a custom savings target
- Track real-time savings progress
- View completed and remaining amounts
- Percentage-based completion tracking

## Progress Experience
- Animated circular progress ring
- Weekly savings visualization
- Smooth progression feedback
- Minimal and focused layout

## Milestone System
- Dynamic savings milestones
- Visual progress checkpoints
- Reward-style progression tracking
- Motivational savings journey

## Premium UI Design
- Dark premium aesthetic
- Calm visual hierarchy
- Consistent spacing and typography
- Refined card-based interface
- Minimal visual clutter

## Motivation Focused
- Designed around behavioral consistency
- Encourages gradual savings habits
- Focused progression instead of information overload

# Philosophy

Monevo is intentionally designed to stay focused and minimal.

The goal is not to become another bloated finance superapp packed with aggressive charts, ads, subscriptions, and unnecessary complexity.

The app prioritizes:
- clarity
- calmness
- consistency
- motivation
- emotional progress
